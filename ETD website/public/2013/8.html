Computer aided system using fuzzy logic for the ability of preschooler

Abstract
Computer controlled systems are the greatest interest in today's world.Examination can be held by means of the paper writing but also computer testing.This system is to classify the ability of the preschoolers.When the child needs to test the exam.There are five kinds of questions in examination:choose the right alphabetic,color,shape,number and mathematical computing .The exam mark is 75 marks.The teacher needs to give other 25 marks to the child depending assessment factor on these the child's attendance,cooperation with the other ,interest in the class and activity in sport.The total marks are 100 marks.As a result,the child will get the Grade of the test depending on their marks .The ability of the children will be determined by using Fuzzy logic.Fuzzy logic is one of the lastest intelligent control techniques.The input is the mark of test and output is the result of test.Both input and the output are analyzed by Fuzzy Decision Making.This system has been implemented by using C# programming language.
